# Waypoint

Waypoint is a Next.js app built with React, Tailwind CSS, and Supabase.

## Requirements
- Node.js 20+
- npm 10+

## Getting Started
```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scripts
- `npm run dev` Start dev server
- `npm run build` Build for production
- `npm run start` Start production server
- `npm run lint` Run ESLint

## Environment
Copy `.env.example` to `.env.local` and fill in required values.

## Deployment
Use your preferred platform (Vercel, Render, etc.). Run `npm run build` and `npm run start` for production.
