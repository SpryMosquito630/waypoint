# Contributing

Thanks for contributing to Waypoint.

## Setup
- Node.js 20+
- npm 10+

```bash
npm install
npm run dev
```

## Workflow
- Use feature branches off `main`.
- Keep PRs focused and small.
- Run `npm run lint` and `npm run build` before opening a PR.

## Commit Messages
- Use clear, imperative messages (e.g., "Add task filters").

## Issues
- Include steps to reproduce and expected/actual behavior.
